Данный проект реализует набор функций для вычисления геометрических параметров различных фигур. Программа предоставляет консольный интерфейс для работы с четырьмя основными геометрическими фигурами.

Проект разработан в рамках лабораторной работы по изучению платформы GitHub и практическому использованию Git для хранения, обновления и распространения исходного кода.
